# Ernest Gitonga — Professional Portfolio

A premium, minimalist portfolio showcasing expertise in Full-Stack Web Development (MERN), Photography, and Videography. Built with a focus on clean UI/UX, professional storytelling, and robust technical execution.

---

## 🗂️ Project Structure
Personal-Portfolio/
├── index.html       → Home page & Dynamic typing animation
├── services.html    → Service offerings & Optimized SVG iconography
├── skills.html      → Technical proficiency & Progress visualization
├── education.html   → Academic history (Starehe Boys' Centre focus)
├── experience.html  → Professional & Volunteer history (Parliament VSS)
├── contact.html     → International-ready lead generation form
├── style.css        → Core design system & Responsive layout logic
└── main.jpg         → Professional profile asset

## ✨ Features

- **Dynamic Typewriter Effect** — Engaging hero section highlighting a multi-disciplinary skill set.
- **Parliament VSS Strategic Placement** — Showcases high-level civic engagement and upcoming legislative service.
- **Global Form Validation** — Enhanced `tel` and `email` logic to ensure accessibility for international clients.
- **Glassmorphism Design** — Modern frosted-glass header effects with throttled scroll performance.
- **Responsive Timelines** — Specialized CSS grid and flexbox logic for a seamless education and experience history.
- **Starehe-Standard Detail** — Comprehensive breakdown of academic excellence and leadership roles.

## 🧠 Concepts Applied

### HTML5 & Logic
| Concept | Where it's used |
|---------|----------------|
| **Pattern Attributes** | Global phone validation logic in `contact.html` |
| **Semantic Architecture** | SEO-optimized tags (`article`, `section`, `nav`) across all pages |
| **SVG Management** | Scalable, lightweight vector icons in `services.html` |

### CSS3 & Design
| Concept | Where it's used |
|---------|----------------|
| **Root Variables** | Centralized color palette control (`--red`, `--gold`, `--bg`) |
| **Bézier Transitions** | Smooth hover states and custom timeline animations |
| **Media Queries** | Mobile-first adjustments ensuring 100% readability on all devices |

### JavaScript
| Concept | Where it's used |
|---------|----------------|
| **requestAnimationFrame** | Throttled scroll performance for the glassmorphism header |
| **Array Iteration** | Typing animation logic for the dynamic hero sub-headline |

## 🖼️ Assets & Design Philosophy

- **Main Profile**: Professional imagery via `main.jpg`.
